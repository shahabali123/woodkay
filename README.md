# WoodKay - Ultimate Free DIY Woodworking Plans & Furniture Tutorials

## Welcome to WoodKay

**WoodKay** is a premier open-source platform dedicated to empowering makers, DIY enthusiasts, and homeowners with high-quality, **free woodworking plans** and comprehensive **furniture building tutorials**. WoodKay aims to democratize woodworking by providing accessible, easy-to-follow guides for building professional-grade furniture at a fraction of the retail cost.

Whether you are looking to build a **rustic farmhouse table**, a **modern floating shelf**, or a complete **outdoor patio set**, WoodKay serves as your digital workshop companion. Our platform is built with modern web technologies to ensure a fast, mobile-responsive experience, making it easy to view plans right from your workshop or garage.

## üõ†Ô∏è Installation & Local Development

Want to contribute to WoodKay or run your own instance? Follow these steps to get the woodworking platform running on your local machine.

### Prerequisites
-   Node.js (v18 or higher)
-   npm or yarn

### Steps

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/woodkay.git
    cd woodkay
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run the Development Server**
    ```bash
    npm run dev
    ```
    Open your browser to `http://localhost:5173` to see the site in action.

4.  **Build for Production**
    ```bash
    npm run build
    ```
